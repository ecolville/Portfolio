Ice Age Trail Educator Map

Capstone project for my MS in Cartography & GIS: an educator-focused interactive geologic map designed to support place-based learning along Wisconsin’s Ice Age Trail. The application connects trail segments to glacial and bedrock geology, points of interest, and teaching-ready interpretive content.

I designed the map experience with a user-centered workflow in mind, emphasizing clear visual hierarchy, accessible design choices, and a narrative structure that helps non-specialists understand geologic context. The final product integrates interactive map exploration with supporting information that translates geologic data into classroom-friendly language.

Skills: ArcGIS Pro/Online, StoryMaps, web cartographic design, UI/UX for education, interactive mapping, geologic interpretation, visual storytelling

UW System Campus FTE (1973–2022)

Interactive Leaflet map created in graduate coursework to explore how UW System full-time equivalent (FTE) enrollment patterns changed over time across campuses. The map uses proportional symbols to communicate relative enrollment magnitude and a time slider to support temporal comparison.

I implemented a complete interactive workflow including loading and parsing GeoJSON attributes, calculating summary statistics for symbol scaling, and dynamically updating symbol sizes and legend values as the time sequence changes. The design emphasizes readable symbology, clear annotation, and an interface that supports exploration without overwhelming the user.

Skills: JavaScript, Leaflet, GeoJSON, proportional symbol mapping, interactive legends, UI controls, data-driven cartography

Food Insecurity Visualization (D3)

Data visualization project built with D3 to explore spatial patterns and change over time in food insecurity across the United States. The goal was to translate a complex dataset into an interactive visual story that supports comparison, pattern recognition, and interpretation.

I designed and implemented the full visualization pipeline: cleaning/structuring the data for use in the browser, binding data to SVG elements, building scales and legends, and adding interaction to support exploration (such as hover states, tooltips, and/or filtering depending on the final implementation). This project strengthened my ability to communicate uncertainty and regional variation using visual encodings that are intuitive for non-expert audiences.

Skills: JavaScript, D3.js, SVG-based design, data joins, interaction design, chart accessibility, visual storytelling

Madison Bikes – Repair Station Finder

Final project for UW–Madison GEOG 576, developed collaboratively with Jessica Steslow. This project implements a full-stack web application to locate bicycle repair stations in Madison using open municipal data.

I worked on backend server logic written in Go and deployed using Google App Engine, enabling data communication between an external API, a managed server environment, and the front-end web application. The project emphasized API workflows, server deployment, and user-centered interface design.

Skills: Visual Studio Code, Go, Google App Engine, full-stack web development, API integration, UI/UX design, collaborative development
